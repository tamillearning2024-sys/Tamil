# 🎓 Tamil Learning Web App

A modern **AI-powered Tamil Learning Platform** designed for students and admins to manage learning, tests, and progress efficiently.

---

## 🚀 Live Demo

👉 https://tamillearning2024-sys.github.io/Tamil/

---

## 📌 Features

### 👨‍🎓 Student

* Dashboard with progress tracking
* View Units & Study Materials (PDF)
* Attend Tests (MCQ-based)
* View Reports & Scores
* Task Management
* Notes Section
* Profile View

### 🧑‍💼 Admin

* Admin Dashboard with analytics
* Add / Edit / Delete Students
* Assign Units & Study Materials
* Create Tests (MCQ system)
* Track Student Performance
* Reports with sorting (A → Z)
* Manage Tasks

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/tamillearning2024-sys/Tamil.git

# Go to project folder
cd Tamil

# Install dependencies
npm install

# Start development server
npm run dev
```

---

## 🚀 Build & Deploy

```bash
# Build project
npm run build

# Deploy to GitHub Pages
npm run deploy
```
